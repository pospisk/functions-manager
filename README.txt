== Description ==

The plugin is used to manage the functions.php file and to make it easier to edit in general.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `functions-manager.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your functions.php file
